# ML Corpus Stats

`ml-corpus-stats` — сервис для статистического анализа языковых корпусов.  
Сервис позволяет вычислять различные статистические параметры текста:  
- Законы Ципфа и Хипса  
- Индексы Юла, Симпсона, Гердана  
- Гапаксы (hapax legomena)  
- Другие лексические показатели  

Сервис разработан с применением принципов ООП: инкапсуляция, наследование, полиморфизм, расширяемость.

## ⚙️ Основные компоненты

### Core
- **User** — пользователь сервиса  
- **Task / TaskStatus** — задача анализа текста  
- **Prediction** — результат анализа  
- **Transaction** — история транзакций пользователя  

### Models
- **BaseModel** — абстрактный интерфейс ML/статистической модели  
- **ZipfModel / HeapsModel / DiversityModel** — конкретные реализации моделей  

### Metrics
- **BaseMetric** — базовый класс для метрик  
- Конкретные метрики: `ZipfMetric`, `HeapsMetric`, `YuleMetric`, `SimpsonMetric`, `HerdianMetric`  

### Services
- **CorpusService** — управление текстовыми корпусами и токенизация  
- **MLService** — запуск моделей по имени  
- **AnalyticsService** — выполнение задач анализа, управление статусом и результатами  

### Utils
- **Tokenizer** — вспомогательная токенизация текста

---

## 📝 Принципы проектирования

- **Инкапсуляция** — все поля приватные, доступ через методы и свойства  
- **Наследование** — `BaseModel` и `BaseMetric` → конкретные модели и метрики  
- **Полиморфизм** — единый интерфейс `analyze()` / `calculate()` для любых моделей и метрик  
- **Расширяемость** — легко добавлять новые метрики и модели без изменения существующего кода
